This release focuses on reliability and polish, with iCloud sync fixes, smoother authentication, and broad quality-of-life improvements across the app.

## Improvements and Fixes

- iCloud sync related bug fixes and performance improvements for more reliable data syncing across devices
- Improved app reliability and overall responsiveness across key screens
- Refined networking and background refresh behavior for more consistent updates
- Polished authentication and session handling for smoother startup and sign-in continuity
- Improved interactions on posts and activities, including likes and comment flows
- Standardized and cleaned up several views for a more consistent in-app experience
- Enhanced cache management with more thorough cleanup, reclaiming more disk space when clearing caches
- Reduced internal complexity with broad code quality and maintenance improvements
