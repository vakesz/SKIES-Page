This release introduces multi-site support, Microsoft Teams chat integration, feed refresh timestamps, and major improvements to macOS optimization, iCloud sync, and overall stability.

## New Features

- **Multi-site support**: SKIES can now connect to different sites, not just one. Organizations can configure custom endpoints via SiteConfig with support for no-auth or MSAL authentication
- **Teams chat**: Open a Microsoft Teams chat directly from user profiles when your organization uses Microsoft sign-in
- **Feed timestamps**: Feeds now show when they were last refreshed
- **Search photos**: Profile photos now appear in search results
- **Open-source licenses**: View the app's dependencies and their licenses in Settings > About
- **Mac keyboard shortcuts**: Use Cmd+, to open Settings on macOS

## Improvements

- **Optimized for Mac**: Avatars, icons, buttons, and content widths now scale properly when running on macOS
- **iCloud sync**: Sync status now updates correctly and persists across restarts
- **Content display**: Posts, comments, and activities look cleaner and scroll more smoothly
- **Pull-to-refresh**: Refreshing feels more reliable with less stale content
- **Browser preference**: Web links now open in your chosen browser, either in-app or in the system browser
- **Sharing**: Shared links work more reliably with your organization's setup
- **Demo mode**: Switching between demo and real accounts after logging out is more reliable. Read history may reset once after this update
- **Activity participants**: Clearer display of who is participating in activities
- **General polish**: Smoother profile loading, better read tracking, and various interaction fixes

## Bug Fixes

- Fixed a crash that could occur when updating the app
- Fixed offline mode incorrectly triggering a sign-out
- Fixed recovery from invalid local data
- Fixed deprecated launch storyboard configuration
